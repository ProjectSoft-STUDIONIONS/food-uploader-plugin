Плагин «Food File Uploader» для Joomla! CMS

Плагин food-uploader-plugin для WordPress CMS. Обслуживание директории food сайта школы. Актуально для сайтов школ РФ.

Описание

Плагин food-uploader-plugin для WordPress CMS разработан для обслуживание директории food на сайтах школ Российской Федерации.

Установку плагина нужно провести через административную панель WordPress CMS.

Скачать последнюю версию плагина для своей версии Joomla! CMS.

Перейти на страницу установки плагинов . Дождаться конца установки компонента.

Плагин установлен. Вам осталось только активировать плагин. Для этого нужно перейти в Плагины и нажать пункт Активировать в блоке плагина.

Директория food присутствует всегда. Для дополнительных директорий, например индивидуального питания, нужно перейти в Настройки компонета. Кнопка настройки плагина присутствует на странице плагина Меню ежедневного питания вправом верхнем углу.

Вы можете перечислить директории, которые нужно создать. Так же вы можете настроить автоудаление старых файлов, при этом вы можете выбрать временной интервал от одного года до пяти лет. Т. е. если выбран один год, то вы сохраните файлы за текущей год и за прошлый включительно.

Данная опция позволяет вам сэкономить объем используемого пространства выделенного системой вашего хост провайдера.

Удачного использования компонента!
